# Silent Below

Developed with Unreal Engine 5

An individual project made from inspiration of games like Metro or Stalker, with it's dark environment. The player finds themselves stuck inside a tunnel sewage system in an post-apocalyptic world, from which they have to escape in order to survive. Watch out, there are mutants in the tunnels with the player.

Control Scheme -> 
WASD - movement
Left Mouse - shoot
R - reload
CTRL - slow walk
SHIFT - run
E - interact
